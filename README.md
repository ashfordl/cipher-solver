# Cipher Solver

###### Created by Luke Ashford and George Wright

A series of tools and libraries written in C# to aid with cryptography and cryptanalysis. 

### Curently Implemented
- Railfence cipher
- Caesar shift
- Columnar transposition cipher
- Polyalphabetic cipher
- Analysis tools:
  - Frequency analysis
  - Bigrams
- GUI applications:
  - Manual solving
